# Working with Lists in Python

## Introduction

Under the guidance of [Dr. Bilal Ahmad](https://facebook.com/example), we learned about Lists, one of Python's most useful data structures.

Lists allow programmers to store multiple values in a single variable and manipulate them efficiently.

## Creating Lists

```python
students = ["Ali", "Ahmed", "Saad"]
```

## Accessing Elements

```python
print(students[0])
```

## Adding Elements

```python
students.append("Usman")
```

## Removing Elements

```python
students.remove("Ahmed")
```

## Common List Operations

```python
numbers = [4,2,7,1]

numbers.sort()
print(numbers)
```

## Applications

Lists are widely used in:

- Student Management Systems
- Data Analysis
- Web Applications
- Machine Learning Projects
- Inventory Systems
