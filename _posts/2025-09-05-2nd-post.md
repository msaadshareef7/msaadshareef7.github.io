## Core Concepts Learned

### Variables

Variables store information.

Example:

```python
name = "Saad"
age = 18
```

### Data Types

Python supports:

- Integer
- Float
- String
- Boolean

Example:

```python
roll_no = 129
cgpa = 3.5
name = "Saad"
is_student = True
```

### Input and Output

Example:

```python
name = input("Enter Name: ")
print(name)
```
