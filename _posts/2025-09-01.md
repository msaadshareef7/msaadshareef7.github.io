# Introduction to Dr. Bilal Ahmad and My First Semester Journey

## Overview

The beginning of my BS Computer Engineering journey marked an important milestone in my academic life. During the first semester, I studied several foundational courses including Programming Fundamentals (PF), Calculus, Discrete Mathematics, and Application of Internet Communication Technology (AICT).

Among all these subjects, Programming Fundamentals quickly became my favorite because it introduced me to the world of programming and problem-solving.

## Meeting Dr. Bilal Ahmad

Programming Fundamentals was taught by **Dr. Bilal Ahmad**, a respected academic and researcher in the fields of Artificial Intelligence (AI) and Machine Learning (ML).

During our introductory classes, Dr. Bilal Ahmad explained:

- Importance of programming in engineering
- Future opportunities in AI and ML
- Real-world applications of Python
- Industry demand for software development skills
- Importance of practical learning

His enthusiasm for Artificial Intelligence inspired many students to explore programming beyond classroom requirements.

## Why Python?

One of the first discussions in class was about Python and its significance.

Python is widely used in:

- Artificial Intelligence
- Machine Learning
- Data Science
- Automation
- Web Development
- Cybersecurity
- Research

Dr. Bilal Ahmad emphasized that Python is one of the most beginner-friendly languages while also being powerful enough for advanced AI systems.

## Semester Goals

At the start of the semester, I set several goals:

1. Learn Python programming.
2. Understand programming logic.
3. Build problem-solving skills.
4. Improve coding confidence.
5. Explore Machine Learning concepts.

## What I Expect to Learn

Throughout this semester, I aim to:

- Master programming fundamentals.
- Learn structured coding practices.
- Build mini projects.
- Understand the basics of AI and ML.
- Prepare for advanced computer engineering courses.

## Reflection

The first few weeks established a strong foundation for my academic journey. Dr. Bilal Ahmad’s guidance motivated me to approach programming not just as a course requirement but as a valuable lifelong skill.
