As we progressed through Programming Fundamentals, Dr. Bilal Ahmad introduced us to one of the most important concepts in software development: Functions.

## What Are Functions?

Functions are reusable blocks of code designed to perform specific tasks.

### Example

```python
def greet():
    print("Welcome to Programming Fundamentals")
```

Calling the function:

```python
greet()
```

## Functions with Parameters

```python
def greet(name):
    print("Hello", name)

greet("Saad")
```

## Functions with Return Values

```python
def add(a, b):
    return a + b

print(add(5,10))
```

## Benefits of Functions

- Code Reusability
- Better Organization
- Easier Maintenance
- Faster Development
- Improved Readability
