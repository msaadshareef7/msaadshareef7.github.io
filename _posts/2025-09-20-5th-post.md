# Loops and Repetition Structures in Python

## Introduction

One of the most important concepts in programming is repetition.

Instead of writing the same code repeatedly, loops allow tasks to be executed automatically multiple times.

## Why Loops Matter

Without loops:

```python
print("Hello")
print("Hello")
print("Hello")
print("Hello")
print("Hello")
```

With loops:

```python
for i in range(5):
    print("Hello")
```

## For Loop

Example:

```python
for i in range(1, 6):
    print(i)
```

Output:

```text
1
2
3
4
5
```

## While Loop

Example:

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

## Nested Loops

Example:

```python
for i in range(3):
    for j in range(3):
        print(i, j)
```

## Practical Applications

Loops are used in:

- Data processing
- Machine Learning
- Web applications
- Automation scripts
- Scientific calculations

## Classroom Exercises

We practiced:

- Multiplication tables
- Number patterns
- Sum calculations
- Factorial programs

Example:

```python
num = 5
fact = 1

for i in range(1, num + 1):
    fact *= i

print(fact)
```
