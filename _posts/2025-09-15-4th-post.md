# Conditional Statements in Python

## Introduction

Programming becomes powerful when a computer can make decisions.

Conditional statements allow programs to choose actions based on specific conditions.

## If Statement

Example:

```python
age = 18

if age >= 18:
    print("Eligible")
```

## If-Else Statement

Example:

```python
marks = 45

if marks >= 50:
    print("Pass")
else:
    print("Fail")
```

## If-Elif-Else Statement

Example:

```python
marks = 80

if marks >= 90:
    print("A Grade")
elif marks >= 80:
    print("B Grade")
else:
    print("C Grade")
```

## Importance

Conditional statements are used in:

- Login systems

- ATM software
- Mobile applications
- AI decision systems
